# 🛠️ GM-ARCHIVE (v1.0)

Надёжная и оптимизированная инфраструктура SQL для мода Garry, предназначенная для управления несколькими серверами на одной базе данных с высокой степенью безопасности посредством триггеров.

## ✨ Ключевые функции

* **Мультисервер** Управляйте всеми своими серверами (Sandbox, DarkRP, TTT) через централизованную структуру.
* **Native Security** - SQL-триггеры для предотвращения автоматического запрета SuperAdmins.
* **Производительность** : Использование сохраненных процедур для снижения нагрузки со стороны Lua.
* **Полный аудит** : автоматические журналы административных действий и чата.
* **Analytics** - Предварительно настроенные SQL-представления для лидеров и онлайн-игроков.

## 📂 Содержимое проекта

1. `01_schema.sql` - Создание таблиц, индексов и взаимосвязей (InnoDB/utf8mb4).
2. `02_seed.sql` Начальные данные (стандартные рейтинги, выборочные серверы).
3. `03_procedures_security.sql` : Системная разведка (функции join/leave, запреты, безопасность).

## 🚀 Быстрая установка

1. Импортируйте файлы в числовом порядке в менеджере MySQL (HeidiSQL, phpMyAdmin).
2. Настройте свой доступ в `03_procedures_security.sql`.
3. Подключите ваш сервер GMod через модуль **MySQLOO*.
